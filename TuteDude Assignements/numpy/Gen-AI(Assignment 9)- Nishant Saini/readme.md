## How to Run

1. Make sure Python is installed.
2. Install NumPy if it is not already installed:

```bash
pip install numpy
```

3. Run any task file using:

```bash
python Task1.py
```

Replace `Task1.py` with the desired task file (`Task2.py`, `Task3.py`, etc.).

---

## Requirements

- Python 3.x
- NumPy

Install NumPy using:

```bash
pip install numpy
```
