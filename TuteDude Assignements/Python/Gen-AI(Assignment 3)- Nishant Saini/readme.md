# Assignment 3 - Functions, Recursion and Functional Programming in Python

## Overview

This assignment focuses on applying Python functions and exploring functional programming concepts such as recursion, lambda functions, map(), and filter().

## Tasks Completed

### Task 1: Discount Function
- Created a function to calculate discounted prices.
- Used default arguments.
- Applied validation rules for discounts.

### Task 2: Recursive Factorial Calculator
- Implemented recursion to calculate factorials.
- Handled edge cases such as zero and negative numbers.

### Task 3: Lambda Functions
- Used lambda expressions for concise calculations.
- Applied GST and discount calculations using anonymous functions.

### Task 4: Using map()
- Applied transformations to a list of prices.
- Generated updated price lists with GST applied.

### Task 5: Using filter()
- Filtered products based on price conditions.
- Created separate collections for different categories.

### Task 6: Combining Functions, map() and filter()
- Processed product prices through multiple functional programming techniques.
- Demonstrated data transformation and filtering workflows.

### Task 7: Menu-Driven Price Manager
- Built an interactive menu system using functions.
- Added new prices dynamically.
- Calculated average prices.
- Displayed the highest recorded price.
